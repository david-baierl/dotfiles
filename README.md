#  dotfiles

## Requirements

- [git](https://git-scm.com/)
- [gnu stow](https://www.gnu.org/software/stow/)

## Installation

```sh
git clone git@github.com:david-baierl/dotfiles.git
```

```sh
cd dotfiles
```

```sh
stow .
```

## Features

This repository features configurations for the following tools

- zsh
- starship
- git
